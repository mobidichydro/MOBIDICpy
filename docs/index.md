# Documentation for `mobidicpy` python package

MOBIDIC (MOdello di Bilancio Idrologico DIstribuito e Continuo – distributed and continuous hydrological balance model) is a raster-based distributed hydrological model that simulates the water and energy balance of the hydrological cycle at the cell level.

MOBIDICpy is a Python implementation of the MOBIDIC model, originally developed in MATLAB by Castelli et al. (Castelli et al., 2009; Yang et al., 2014).


## Installation

To install mobidicpy from GitHub repository, do:

```console
git clone git@github.com:mobidichydro/mobidicpy.git
cd mobidicpy
python -m pip install .
```
